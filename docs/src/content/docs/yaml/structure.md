title: YAML Configuration (Removed)
description: YAML configuration has been removed from DartUnit. Rules are now written as Dart files.
---

:::caution[This section has been removed]
YAML configuration is no longer part of DartUnit. Rules are written as plain Dart files in the `arch_test/` folder — no YAML files, no configuration format to learn.
:::

If you are looking for how to configure DartUnit, see:

- [Quick Start](/overview/quickstart) — set up and write your first rule in 5 minutes
- [Creating Custom Rules](/custom-rules/creating) — how to write rule files in Dart
- [Rules (ArchitectureRule)](/fundamentals/rules) — the full rule API
- [Built-in Presets](/presets/layer) — ready-made rule sets for common patterns
